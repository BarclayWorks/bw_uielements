# BW UI Elements Field

Joomla custom field plugin to provide organising UI elements in field group tabs.

## Description
Enables organisation and display of custom field UI elements within tabbed field groups, improving content editing interface and user experience.

## Features
- Tab-based organisation of field groups
- Improved custom field layout
- Enhanced editor interface
- Compatible with Joomla 4.x+

## Installation
1. Download the plugin package
2. Install via Joomla Extensions Manager
3. Enable the plugin

1. Create a custom field in Content â†’ Custom Fields
2. Select "BW UI Elements" as the field type
3. Configure tab organisation for your field groups
